# Proyecto API REST - Semana 19 🚀

Este proyecto consiste en el desarrollo de una API REST profesional construida en Node.js utilizando **Express**, **Prisma ORM** y **PostgreSQL**. Cuenta con autenticación y autorización basado en roles
---

## Tecnologías Utilizadas

*   **Node.js & Express:** Entorno de ejecución y framework para la creación de la API.
*   **PostgreSQL:** Base de datos relacional para el almacenamiento seguro de datos.
*   **Prisma ORM:** Mapeador objeto-relacional para la gestión y migración de la base de datos.
*   **JSON Web Tokens (JWT):** Manejo de sesiones de usuario de forma segura y sin estado.
*   **Bcrypt:** Encriptación de contraseñas en la base de datos.
